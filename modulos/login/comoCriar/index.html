O método de registro de docentes, monitores e discentes são diferentes, veja abaixo as condições de cada um

Alunos:

Os dados de alguns alunos já estão disponíveis no sistema. Se este for o seu caso, você irá conseguir efetuar login utilizando este dados:

       Login: sua matrícula
       Senha: ifba.SEU CPF SEM PONTO
Caso não consiga efetuar login desta maneira, seus dados ainda não foram enviados.
Para enviar os seus dados você pode consultar:

  • Seu representante de turma
  • Docentes registrados no SiGAÊ
  • Administradores do sistema

Professores e monitores:

Os dados básicos dos docentes e monitores não estão previamente disponíveis no sistema, sendo obrigatoriamente necessário o pré cadastro destes pelos administradores


Os dados requisitados no registro, tanto de discentes, monitores e docentes, são:

  • Nome completo
  • Matrícula
  • Cadastro de Pessoa Física (CPF)
